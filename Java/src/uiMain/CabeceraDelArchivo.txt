Funcionalidad del módulo: Este módulo tiene como objetivo servir de interfaz entre el usuario y el sistema, generando así las condiciones de interactividad necesarias para proveer un entendimiento claro y conciso de las funcionalidades y las opciones.
Componentes del módulo: En el módulo se encuentra la clase Banquero donde se definen las diferentes opciones y casos que serán posibles en la interfaz, más adelante se da una explicación más a profundidad sobre esta clase.
